1024B Fluid Storage Disk

  1. Storage
  2. 1024B Fluid Storage Disk
On this page
  1. Upgrading
  2. Uncrafting
  3. Storage capacity

Fluids in the storage network can be stored on a 1024B Fluid Storage Disk, which must be placed in a Disk Drive.

Upgrading

You can combine the 1024B Fluid Storage Disk with a higher tier fluid storage part to upgrade it.

The original fluid storage part will be returned.

Uncrafting

Sneak and use an empty 1024B Fluid Storage Disk to retrieve its components:

  • The corresponding fluid storage part

  • Storage Housing

Storage capacity

The 1024B Fluid Storage Disk can store 1024 buckets of any fluid.
